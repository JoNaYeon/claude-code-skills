# 🌅 Morning Briefing

매일 아침 하루를 시작하기 위한 전체 브리핑을 생성합니다.

## 실행 순서

### 1. 📅 오늘의 캘린더 일정
- `gcal_list_calendars`로 캘린더 목록 조회
- `gcal_list_events`로 오늘 날짜(자정~자정) 기준 전체 일정 조회
- 시간순으로 정렬하여 출력
- 형식:
  ```
  ⏰ HH:MM  이벤트 제목
            📍 장소 (있을 경우)
            👥 참석자 (있을 경우)
  ```

### 2. ✅ Google Tasks 할일
- `gcal_list_events`에서 Tasks 캘린더 포함 조회 (Google Tasks는 캘린더에 통합됨)
- 미완료 할일만 필터링
- 마감일 기준 정렬
- 형식:
  ```
  • [ ] 할일 제목 (마감: MM/DD)
  ```

### 3. 📰 주요 뉴스 요약
- `WebSearch`로 오늘 날짜 기준 주요 뉴스 검색
- 검색어: `"오늘 주요 뉴스 {오늘날짜}"`, `"today top news {오늘날짜}"`
- 국내 뉴스 5건 + 해외 뉴스 3건 요약
- 형식:
  ```
  🇰🇷 국내
  1. [제목](링크) — 한 줄 요약
  2. ...

  🌐 해외
  1. [제목](링크) — 한 줄 요약
  2. ...
  ```

### 4. 📧 이메일 요약
다음 두 계정의 오늘 수신 메일을 요약합니다:
- `nannanijo@gmail.com`
- `nayeon@seemedi.com`

- `gmail_search_messages`로 각 계정 `after:{어제날짜}` 조건 검색
- 읽지 않은 메일 우선, 중요도 높은 순 정렬
- 형식:
  ```
  📬 nannanijo@gmail.com
  • [발신자] 제목 — 한 줄 요약

  📬 nayeon@seemedi.com
  • [발신자] 제목 — 한 줄 요약
  ```

### 5. 🔁 이전 미완료 항목 확인

오늘 할일 페이지 작성 전, 직전 task 파일을 읽어 미완료 항목을 오늘로 이월합니다.

#### 실행 방법
1. `C:\Users\user\OneDrive\vault\03_daily\task\` 폴더 내 파일 목록을 날짜 기준으로 정렬
2. 오늘 날짜보다 이전인 파일 중 가장 최근 파일을 찾음
3. 해당 파일에서 `- [ ]` (미완료) 항목만 추출
4. 이월 항목 없으면 → 이 단계 스킵
5. 이월 항목 있으면 → Step 6의 할일 페이지에 `## ⏰ 이월된 항목` 섹션으로 추가

#### 이월 항목 형식
```
## ⏰ 이월된 항목
- [ ] 항목 내용 (📅 원래 날짜: MM/DD)
- [ ] 항목 내용 (📅 원래 날짜: MM/DD)
```

#### 이월 규칙
- `- [x]` (완료) 항목은 이월하지 않음
- 이월 항목은 원래 날짜를 `(📅 원래 날짜: MM/DD)` 형식으로 반드시 표기
- 이월 항목이 오늘 캘린더/Tasks와 중복될 경우 중복 제거
- 직전 날짜 파일 1개만 확인 (연쇄 이월 없음)

### 6. 📝 Obsidian 할일 페이지 작성

브리핑 생성 후, 캘린더·Tasks 데이터 및 이월 항목을 반영하여 Obsidian에 오늘의 할일 페이지를 작성합니다.

#### 파일 경로
```
C:\Users\user\OneDrive\vault\03_daily\task\YYYY-MM-DD.md
```

#### 중요도 분류 기준

| 항목 | 중요도 |
|------|--------|
| 약속, 회의, 미팅 (시간이 지정된 일정) | 🔴 높음 |
| Google Tasks 할일 (마감일 있음) | 🔴 높음 |
| 업무 관련 일정 (날짜 지정됨) | 🔴 높음 |
| 업무 관련 항목 (날짜 없음) | 🟡 낮음 |
| 생일, 기념일, 공휴일 등 | 🟡 낮음 |
| 마감일 없는 할일 | 🔵 날짜 미정 (별도 분류) |

#### 카테고리 분류 기준
- **개인**: 개인 약속, 생일, 여가, 건강, 생활 관련
- **업무**: 회사/프로젝트 관련 일정 및 Tasks
- **공부**: 스터디, 강의, 학습 관련

#### 페이지 형식

```markdown
---
tags: [daily, task]
date: YYYY-MM-DD
---

# YYYY년 MM월 DD일 (요일)

## ⏰ 이월된 항목
- [ ] 항목 내용 (📅 원래 날짜: MM/DD)

## 🔴 오늘 반드시 할 일

### 개인
- [ ] 항목 (HH:MM, 📍 장소)

### 업무
- [ ] 항목 (HH:MM)

### 공부
- [ ] 항목

## 🟡 여유 있으면 할 일

### 개인
- [ ] 항목

### 업무
- [ ] 항목

## 🔵 날짜 미정 (언젠가 해야 할 것들)
- [ ] 항목
```

#### 처리 규칙
- `C:\Users\user\OneDrive\vault\03_daily\task\` 폴더가 없으면 먼저 생성
- 이월 항목이 없으면 `## ⏰ 이월된 항목` 섹션 생략
- 오늘 날짜 파일이 이미 존재하면 덮어쓰지 않고, 기존 내용 아래 `---` 구분선 후 업데이트 내용 추가
- 내용이 없는 섹션(해당 카테고리 항목 없음)은 생략
- 시간이 있는 일정은 시간 표기 포함

## 주의사항
- 오늘 날짜는 항상 실행 시점 기준으로 계산
- 일정이 없으면 "오늘 일정 없음" 출력
- 메일이 없으면 "새 메일 없음" 출력
- 뉴스 링크는 반드시 원문 URL 포함
- Obsidian 파일 저장 후 브리핑 마지막에 저장 경로 안내
